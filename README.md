# IDAI105-2505467--SOHA-PATEL.
SmartCharging Analytics is an interactive EV charging station dashboard built with Streamlit. It applies K-Means clustering, Apriori association rule mining, and anomaly detection (Z-score, IQR, Isolation Forest) on 500 global stations. Features include demand heatmaps, cluster maps, EDA visualizations, and live filtering.
